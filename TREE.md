# TREE.md — Estrutura do Repositório

Abaixo está a estrutura do repositório do projeto, com descrições curtas de cada arquivo/pasta.

```
project-root/
├─ README.md                     # Descrição geral do projeto (já gerado)
├─ RELATORIO-ACADEMICO.md        # Relatório técnico em formato ABNT
├─ TEMPLATE-SCREENSHOTS.md       # Guia padronizado de capturas de tela
├─ LOG-EXAMPLES.md               # Exemplos de logs fictícios para análise
├─ PLAYBOOK-INCIDENT-RESPONSE.md # Checklist e procedimentos de resposta a incidentes
├─ TREE.md                       # Estrutura do repositório (este arquivo)
├─ safe-demos/                   # Scripts e demos seguros (didáticos)
│   ├─ safe_demos.py             # Scripts Python seguros de demonstração
│   ├─ simulated_ransom/         # Arquivos de exemplo para simulação de "ransom"
│   │   ├─ README.md             # Instruções de uso e cautelas
│   │   └─ files/                # Arquivos de teste (pequenos, inofensivos)
│   └─ simulated_keylogger/      # Inputs e outputs simulados para keylogger
│       ├─ README.md             # Instruções de uso (não captura real)
│       ├─ input_simulado.txt    # Sequência fictícia de teclas
│       └─ registro_simulado.txt # Resultado do processamento
├─ docs/                         # Documentação complementar e anexos
│   ├─ ANEXOS.md                 # Indicação de anexos (screenshots, logs)
│   └─ GLOSSARIO.md              # Termos técnicos e definições
```
